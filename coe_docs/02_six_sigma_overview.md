# Six Sigma Overview (COE Reference)

Six Sigma is a structured methodology used to reduce process variation and defects using data.

DMAIC process:
Define – Identify the problem and scope  
Measure – Collect baseline data  
Analyze – Identify root causes  
Improve – Implement solutions  
Control – Sustain improvements with monitoring

COE applications:
- deviation reduction
- batch record accuracy
- complaint cycle time improvement
- quality compliance metrics
