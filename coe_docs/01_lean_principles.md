# Lean Principles (COE Reference)

Lean is a continuous improvement approach focused on maximizing customer value while minimizing waste. In a COE context, Lean is used to simplify workflows, reduce handoffs, and eliminate rework.

Common wastes include:
- Waiting: approvals or idle queues
- Defects or rework
- Over-processing
- Excess movement of information
- Work-in-progress backlog

Key Lean methods used in COE programs:
- Value Stream Mapping
- Standard Work
- Kaizen improvements
- Visual management dashboards
- Root cause analysis

Example:
If deviation closures are delayed, Lean helps identify bottlenecks, simplify approvals, and standardize documentation.
